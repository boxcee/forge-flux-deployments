## Integration Defect Detail

**ArgoCD source value mismatch**

- `src/index.js` line 121: `const logParams = { source: 'argo' };`
- `src/frontend/index.jsx` line 127: Select option `{ label: 'Argo', value: 'argocd' }`
- **Impact:** When user selects "Argo" filter, both `getEventStats` and `getEventLog` receive `source: 'argocd'`, but all ArgoCD events are stored with `source: 'argo'`. SQL `WHERE source = 'argocd'` returns 0 rows.
- **Fix:** Change `src/index.js` line 121 to `source: 'argocd'` (+ update test assertion), OR change frontend filter value to `'argo'`.
- **Affected requirements:** UI-02, UI-03
