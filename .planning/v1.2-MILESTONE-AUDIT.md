## Tech Debt

2 minor items across 2 phases. Neither is a blocker.

### Phase 8: Admin UI Event Log Tab
- `isLoading={false}` hardcoded on DynamicTable — loading overlay absent during source filter re-fetches. Spinner covers initial load only. UI polish gap, not functional.

### Phase 7: Event Log Backend
- `getStats()` SQL enumerates specific HTTP codes (400, 401, 502, 503) rather than ranges. Correct for all current handler exit codes, but would silently exclude future status codes from the "failed" count.
