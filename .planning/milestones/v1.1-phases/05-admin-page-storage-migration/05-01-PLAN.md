
Resolver function keys (invoked via @forge/bridge invoke()):
- 'getConfigStatus' -> returns { flux: { configured }, argocd: { configured } }
- 'getWebtriggerUrls' -> returns { flux: string, argocd: string }
- 'setFluxSecret' -> payload: { secret: string } -> returns { success: boolean, error?: string }
- 'setArgoSecret' -> payload: { token: string } -> returns { success: boolean, error?: string }
- 'deleteFluxSecret' -> returns { success: boolean }
- 'deleteArgoSecret' -> returns { success: boolean }
