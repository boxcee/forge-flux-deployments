       - Add new test: "returns 503 when flux secret not configured" -- mockGetFluxSecret returns undefined
       - Add new test: "returns 503 when argo secret not configured" -- mockGetArgoSecret returns undefined
       - Verify 503 response body contains "not configured" message
       - Keep all existing test cases (they should work with mocked storage returning the same values)
